        장점: Random access(임의 접근).
        단점: 데이터를 중간에 삽입/삭제하는데 이동하는 데이터가 겁나 큼(overhead 큼)

        2>리스트(단일,이중,원형): 원소(노드) 하나하나가 따로 저장됨.(청크형태X)

        장점: 데이터를 중간에 삽입/삭제하기 편함.
        단점: 각각의 노드마다 포인터 변수 필요(==메모리 많이 사용). Random access 불가// 데이터에 접근할일 많으면 리스트 보단 배열!
