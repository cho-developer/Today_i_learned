    4)pass by value , pass by reference
	Pass by Value (값에 의한 호출): 변수의 '값' 자체를 복사하여 함수로 전달.
	함수 내부에서 값을 변경해도 원본 변수에는 전혀 영향XX.

	Pass by Reference (참조에 의한 호출): 변수의 '메모리 공간(참조)' 자체를 직접 전달.
	원본 변수와 매개변수가 완전히 동일한 메모리를 가리키므로, 함수 내부의 변경이 원본에 직접적으로 반영.

    15)NULL(0) vs EOF(-1)
    15-1 공통 : 둘다 '없음'을 나타내기 위해 도입하는거.

    15-2 NULL : 0. 주소는 0번지 주소(NULL)를 쓰지 않으므로 주소를 다루는 애들(fgets,fopen,malloc 등.)이 없음을 나타내기 위해 !=NULL을 씀.
    15-3 EOF : 주소를 다루는 애들이 0을 사용하지 않는데 반해, 일반적인 문자(ASCII 코드 0~127)다루는 애들(fgetc, getchar, scanf 등)은 0을 씀 . so 없음을 NULL로 해버리면 NULL을 0으로 인식해서 유효한 값으로 인식 .그래서 유효한 문자와 겹치지 않는 EOF(-1) 도입.
