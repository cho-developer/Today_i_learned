0. DP(Dynamic Progamming):  계산된 값을 저장해놨다가 계산된값을 재사용하는것.//분할 정복 알고리즘과 매우 유사.

1. 분할정복 알고리즘과 DP의 차이:
    DP: 이전에 계산한거랑 똑같은걸 또 계산하는 문제에선 DP.
    분할정복:분할 정복시, 반복되는 계산이 없는 문제에선 분할정복(즉, 계산된거 저장X할때).

2. DP방식:
    1)Top-down:큰 문제를 하위문제로 내려가서 해결하기. 
    이때 계산해 놓은걸 배열에 저장하는 방식(memoization) use.
    2)bottiom-up: 하위에서 부터 해서 위로 가기. 반복문 use.
