0.scnaf에서 공백과 스페이스 처리 하는법
	%d, %f, %s: 이 녀석들은 똑똑해서 앞에 있는 공백, 탭, 엔터를 알아서 무시하고 실제 데이터가 나올 때부터 읽음. 
	%c: 문제. 입력 버퍼에 있는 공백이나 엔터도 '하나의 문자'로 취급
	
	e.g.,)
	scanf("%d", &age);   // 20 누르고 엔터 입력 -> 버퍼에 [20] [\n] 들어감
	scanf("%c", &grade); // age가 20을 가져가고 [\n]이 남음. grade는 [\n]을 읽고 입력을 종료해버림!
	해결법)
	scanf(" %c", &grade) //%c 앞에 스페이스 하나 추가하면 버퍼에 남은 \n 완벽 무시

1. scanf 에서 특정문자 입력 안받고 싶을때
	 대입 생략 문자 * 사용하기 (%*c, %*d, %*s 등)
	 입력을 읽어오기는 하지만 변수에 저장하지 않고 그대로 버리고 싶을 때, % 기호와 서식 지정자(c, d, s 등) 사이에 별표(*)를 넣기.

	예시 코드) 
	int a, b;	// "10 X 20" 처럼 입력될 때, 가운데 문자 하나를 무시
	scanf("%d %*c %d", &a, &b); //


2. 문자열 입력 받을시 띄어쓰기 포함해서 한 줄 다 받고 싶을때
	scanf("%[^\n]", str); >> %[^ㅁ]가 %s 대신함.		e.g.,) scanf("%[^\n]") // 버퍼에 남아있으니깐 띄어쓰기는 포함된다고 말한거임. str안에는 \n 직전까지만 들어감.
	// %[^ㅁ],%s 둘다 "여러 글자(문자열)를 읽어온다"는 서식 지정자.but 종료 조건이 다름
	%s: 공백(스페이스바), 탭, 엔터를 만나면 읽기를 멈춤.
	%[^ㅁ]: 오직 ㅁ를 만날 때까지만 읽기를 계속함.
	[^ㅁ]가  데이터를 읽고 ㅁ **앞**에서 멈춤!!!! //즉 , %[^ㅁ] 하면 ㅁ 안들어감.

	2-2: %[]는 %s와 달리 공백을 알아서 무시 못함.
